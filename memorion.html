¿Qué es Memorión?

Memorión es un juego de memoria en el que tienes que voltear cartas y encontrar las parejas lo más rápido posible.

Este proyecto fue desarrollado por estudiantes como ejercicio práctico para aplicar conocimientos de programación y diseño web usando HTML, CSS y JavaScript.
